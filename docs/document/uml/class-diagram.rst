    .. code-block:: rest

       .. uml::

           abstract class  "Abstrakte Klasse"

        abstract class  "Abstrakte Klasse"

``diamond``, ``<>``
    An empty diamond stands for an association, a black diamond for a
    composition.

    Example:

    .. code-block:: rest

       .. uml::

           diamond         Assoziation

    .. uml::

        diamond         Assoziation
